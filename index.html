# Focus Dial A tiny, self-hosted focus timer with a hand-turned brass dial instead of a generic progress ring. No build step, no accounts, no tracking — open `index.html` and go. ![Focus Dial screenshot](screenshot.png) ## Features - **Drag-to-set dial** — turn the knob (mouse, touch, or arrow keys) to choose 1–60 minutes - **Start / pause / resume / reset** a session - A short synthesized chime plays when a session finishes (no audio file needed) - **Today's sessions** are logged locally in your browser (`localStorage`) — nothing is sent anywhere - Fully responsive, keyboard accessible, and respects `prefers-reduced-motion` ## Running it No build tools required. ```bash git clone https://github.com//focus-dial.git cd focus-dial open index.html # or just double-click the file ``` Or serve it locally: ```bash python3 -m http.server 8000 # then visit http://localhost:8000 ``` ## Deploying to GitHub Pages 1. Push this repo to GitHub. 2. Go to **Settings → Pages**. 3. Under "Build and deployment", set **Source** to `Deploy from a branch`, branch `main`, folder `/ (root)`. 4. Your timer will be live at `https://.github.io/focus-dial/`. ## Project structure ``` focus-dial/ ├── index.html # markup ├── style.css # design tokens + styles ├── script.js # dial geometry, timer logic, session log └── README.md ``` ## Tech Plain HTML, CSS, and JavaScript — no framework, no dependencies. Fonts are loaded from Google Fonts (Fraunces, IBM Plex Sans, IBM Plex Mono). ## License MIT — see [LICENSE](LICENSE).
